# Roadmap

## Short Term (1-2 releases)
- Configurable port (OMNITERM_PORT)
- WebSocket authentication (token/header)
- Persistent output buffering & paginated retrieval
- Add unit and integration test coverage

## Mid Term
- Support more asset types: VNC / RDP / Database
- Asset config validation & encryption (private key/password masking)
- AI tool chain plugin system (file inspection / terminal command suggestions)
- Internationalization (i18n) foundation

## Long Term
- Multi-instance collaboration & centralized management
- RBAC permission model
- Audit logs & conversation replay
